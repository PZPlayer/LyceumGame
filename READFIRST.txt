Для игры потребуется:
 Друг(если нет, то Мама)
 Клавиатура
 Позитивный настрой



Управление:
 Для уровня "Кликер":
  для 1 игрока Левый шифтъ
  для 2 игрока Правый контрл

 Для уровня "Танчики":
  для 1 игрока W,A(движение), SPACE(стрелять) 
  для 2 игрока Верхняя/Нижняя стрелочка(движение), правый Enter(стрелять)

 Для уровня "Хокей":
  для 1 игрока W,A(движение)
  для 2 игрока Верхняя/Нижняя стрелочка(движение)
